Desktop slot machine game with player accounts and admin panel.

## Features
- Player registration & login
- Animated reel spin (step-by-step stop)
- Deposit & withdraw system
- Sound effects (toggle on/off)
- Admin panel:
  - Change winning probability
  - Change admin password
  - View & delete players
  - Sort players

## Tech Stack
- C# (.NET Framework 4.8)
- Windows Forms
- LINQ to XML (XML persistence)

## How to Run
1. Open solution in Visual Studio
2. Build the solution
3. Run the project
